# rails-module-ionicons
> Rails module for ionicons.


## Step by step:
+ https://rubygems.org/gems/ionicons-rails
+ use ionicons-rails:
```bash
gem 'ionicons-rails', '~> 2.0'
```
+ add to assets pipe line:
```ruby
 *= require ionicons
```
